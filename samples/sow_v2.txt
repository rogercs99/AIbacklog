1. Objetivo
- Implementar portal de clientes con registro, recuperación de contraseña y onboarding guiado.

2. Alcance técnico
- Hosting en Azure North Europe.
- Base de datos PostgreSQL 15 con réplica.
- API REST y webhooks para integrar con CRM.

3. Compliance
- Cumplimiento ISO 27001 y GDPR.
- Retención de logs 12 meses.

4. Dependencias
- Acceso a entorno de pruebas del cliente confirmado.
- Acceso a sandbox de pagos (pendiente).
